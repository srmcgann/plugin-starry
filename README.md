# KiwiIRC - Starry Plugin

This plugin adds an animated starfield to KiwiIRC's UI. 
Works best with dark themes.


    "starry":{ "color": "#2400a8" },
    "plugins": [
        {"name": "starryPlugin", "url": "./static/plugins/plugin-starry/dist/main.js"} 
    ]
